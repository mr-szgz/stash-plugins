# Reference Index

## Topic folders

- `overview/` - official Stash plugin overview and capabilities
- `config-and-settings/` - general Stash configuration and settings references relevant to plugins
- `plugin-task-systems/` - external plugin tasks and embedded JavaScript plugin tasks
- `ui-plugin-api/` - UI JavaScript plugin API, including `PluginApi.register.route`
- `app-api-and-client-libraries/` - GraphQL API reference and Stash client-library references
- `plugin-sources-and-community/` - plugin/theme/userscript/integration/community indexes and source feeds
- `raw-json/` - GitHub and PyPI metadata snapshots for latest-release and repo references
- `raw-html/` - raw repository page snapshots
